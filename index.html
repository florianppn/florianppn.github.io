MES PROJETS

Mahjong

Jeu du Mahjong (version simplifiée) implémenté en Python avec la bibliothèque graphique Tkinter. Pour ce projet j'ai utilisé l'architecture MVC ainsi que le pattern observateur. J'ai également utilisé le pattern strategy pour les différents types de grilles (losange, rectangle, beignet, ...,).

Taquin

Jeu du Taquin implémenté en Java avec la bibliothèque graphique Swing. Pour ce projet j'ai utilisé l'architecture MVC ainsi que le pattern observateur.

Gridarena

Ce jeu de combat s'appuie sur des mécanismes stratégiques au tour par tour, dans lesquels les joueurs doivent utiliser des tactiques et de la planification pour survivre et éliminer leurs adversaires. Pour ce projet j'utilise l'architecture MVC ainsi que le pattern Observer. Dans ce projet, j'utilise également le pattern Strategy, le pattern Proxy, le pattern Factory Method et le pattern Adapter.
